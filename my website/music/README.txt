# Music Files for Focus Music Player

This folder contains the music files for the Focus Music Player tool. You need to add your own MP3 files with the following filenames:

## Ambient Category
- ambient-study.mp3
- deep-focus.mp3
- calm-mind.mp3

## Nature Category
- forest-sounds.mp3
- ocean-waves.mp3
- rain-sounds.mp3

## Lofi Category
- lofi-study.mp3
- chill-beats.mp3
- lofi-coffee.mp3

Place each file in its corresponding category folder (ambient, nature, or lofi).

## Note
- All files should be MP3 format
- Keep file sizes reasonable (under 5MB per file is recommended)
- Use copyright-free or properly licensed music only
